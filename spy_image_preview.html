对比结论

MAL API ✅ 搜索精准（SPY×FAMILY ID=50265），图片均为官方 Key Visual / 剧照，主题100%相关
⚠️ 下载速度不稳定（偶发 Read Timeout），需重试机制；图片为竖图（~420×600），适合封面不适配横版
Bing 搜索 ⚠️ 关键词依赖强，中文词「同人图」「动漫壁纸」容易召回到素材站/图库
❌ 本次 Bing 结果多来自国内素材聚合站（699pic/douding/xjishu），图片内容质量存疑，需人工确认
推荐策略 MAL API 为主（可指定 MAL ID 绕过关键词搜索），Bing 仅作为补充；可增加黑名单域名（699pic.com, douding.cn 等国内素材站）
